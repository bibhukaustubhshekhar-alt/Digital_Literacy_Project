## 📂 Project Overview
This repository contains my Digital Literacy Portfolio created as part of the CSE0001 Digital Literacy course.

The objective of this project is to develop essential digital literacy skills including:

* Professional online presence
* Digital communication
* Collaboration tools
* Cyber safety awareness
* Responsible digital behaviour

This project contains five tasks corresponding to each module of the course.

## 🛠 Task Modules

### [Task 1: Digital Literacy Awareness Infographic](./task-1-presentation/)
* **Tool Used:** Canva (Modern Steps Infographic template).
* **Topics Covered:** Definition of Digital Literacy, Academic/Collaboration Tools (Google Docs, Canva, Prezi), and Safe Internet Practices.
* **Key Insight:** Balancing visual weight with text was challenging. I simplified the final design to 2-3 bullet points per section to ensure it wasn't "cluttered".

### [Task 2: Student Digital Portfolio](./task-2-portfolio/)
* **Platforms Set Up:**
* **GitHub:** For hosting code and version control. I created a profile README to define my 4-year goals.
* **LinkedIn:** For professional networking; updated with B.Tech details (Expected Grad: 2029).
* **Kaggle:** For data science community engagement and ML micro-courses.
* **Strategy:** I plan to update these every semester to tell a coherent story of growth by my 2028-29 placement year.

### [Task 3: Coding & Collaboration Platforms](./task-3-platforms/)
* **Coding Practice:** Completed the "Solve Me First" Python challenge on **HackerRank**, focusing on arithmetic operators and clean code.
* **Collaboration:** Developed a "Digital Literacy Awareness Quiz" using **Google Forms** to practice data collection and unambiguous questioning.
* **Quiz Link:** (https://docs.google.com/forms/d/e/1FAIpQLSfGivx_P0dnL8WrKYC0tvYOrrhgYGeZZYBRfCj6zFcfHs6sQA/viewform).

### [Task 4: Professional Communication & Etiquette](./task-4-email-etiquette/)
* **Email Correspondence:** Drafted formal requests for a Data Structures assignment extension and a summer internship application at TechNova Solutions.
* **Social Media Ethics:** Created a "Do's and Don'ts" guide. 
    * **Key Rule:** Never post complaints about faculty/college publicly; use private, professional channels instead.

### [Task 5: Cybercrime Awareness & Prevention](./task-5-cybercrime/)
* **Case Study:** Researched a **UPI/Online Payment Fraud** scenario involving a fake "senior student" selling textbooks to first-year students.
* **Prevention Checklist:** Includes setting UPI transaction caps, using 2FA, and avoiding public Wi-Fi for banking.
* **Reporting:** Save the National Cyber Crime Helpline: **1930** and portal: [cybercrime.gov.in](https://cybercrime.gov.in).

## 📈 Personal Reflection
The most unsettling discovery during this project was how precisely cyber-scams are engineered to target specific student needs (like textbooks in August). As a result, I have adopted a strict rule: **No UPI transfer without a verified identity check via an official channel**.
